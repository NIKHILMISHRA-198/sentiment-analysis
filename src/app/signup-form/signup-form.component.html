Privacy Preserving Real Time Sentiment Analysis

This application uses deep learning and convolutional neural networks to provide real time sentiment analysis. Sometimes, during textual conversation (especially for non-English speakers), the tone of the conversation does not convey the actual feeling. The analysis is done by a CNN model (trained on 25000 IMDB reviews) to separate text into positive or negative class. The classification is done in the browser, so the user's chat message does not leave his/her system.
